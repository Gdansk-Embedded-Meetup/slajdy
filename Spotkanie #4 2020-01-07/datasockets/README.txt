Do uruchomienia przykładu potrzebny jest Node.js i menedżer pakietów NPM. Można je pobrać ze strony nodejs.org.

Po instalacji otwieramy terminal w folderze datasocket_test i wywołujemy komendę: npm install ..\datasocket ..\dataplot express
Po zainstalowaniu zależnych pakietów uruchamiamy serwer komendą: node main.js
Możemy teraz w przeglądarce otworzyć stronę pod adresem 127.0.0.1:3000
